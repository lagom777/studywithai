9. 강화학습

🧒 강화학습 = 강아지 훈련

잘하면 간식(보상), 잘못하면 무보상. 계속 시도 → 스스로 최적 전략을 찾는다.

  • AlphaGo = 수백만 번 자기 자신과 바둑 두며 실력 쌓기
  • 게임 AI = 마리오 1억 번 플레이하며 클리어법 터득
  • Q값 = "이 상황에서 이 행동하면 미래에 얼마나 좋아지는가?" 기대 점수
AlphaFold는 강화학습은 아니지만, "50년 난제"였던 단백질 구조 예측을 풀어낸 역사적 사건. 신약 개발 속도를 획기적으로 올림.

강화학습(RL)은 에이전트가 환경과의 상호작용을 통해 누적 보상을 최대화하는 정책을 학습하는 패러다임이다. Markov Decision Process (MDP) $(\mathcal{S}, \mathcal{A}, P, r, \gamma)$에서 목표는: \[ \pi^* = \arg\max_\pi \mathbb{E}\left[\sum_{t=0}^\infty \gamma^t r_t\right] \]

9.1 가치 함수와 Bellman 방정식

상태가치 $V^\pi(s) = \mathbb{E}^\pi[G_t | s_t = s]$, 행동가치 $Q^\pi(s,a)$는 다음을 만족한다: \[ Q^\pi(s,a) = \mathbb{E}[r + \gamma \mathbb{E}_{a'\sim\pi}[Q^\pi(s',a')]] \] 최적 가치는 Bellman 최적방정식의 고정점이다.

9.2 주요 알고리즘

  • Q-Learning: $Q(s,a) \leftarrow Q(s,a) + \alpha[r + \gamma \max_{a'}Q(s',a') - Q(s,a)]$. Tabular 설정에서 최적으로 수렴.
  • DQN(Mnih et al., 2015): 신경망으로 $Q$ 근사, 경험 재현과 타겟 네트워크로 안정화. Atari에서 인간 수준 달성.
  • REINFORCE: $\nabla_\theta J(\theta) = \mathbb{E}[\nabla_\theta \log \pi_\theta(a|s) \cdot G_t]$. 정책 그래디언트의 기본형.
  • Actor-Critic: 정책(actor)과 가치(critic)를 동시 학습. A2C, A3C.
  • PPO(Schulman et al., 2017): 클리핑된 대리 목적함수로 안정적 업데이트. LLM 정렬에 사용.
  • SAC(Haarnoja et al., 2018): 최대 엔트로피 RL. 연속 제어의 표준.

9.3 주요 성과

AlphaGo(Silver et al., 2016)는 MCTS와 딥러닝을 결합해 이세돌을 이겼다. AlphaZero(2017)는 인간 기보 없이 자기대국만으로 바둑·체스·쇼기를 초인 수준으로 정복했다. AlphaStar, OpenAI Five는 실시간 전략·MOBA 게임에서 프로를 이겼다. AlphaFold 2(Jumper et al., 2021)는 단백질 구조 예측의 "50년 문제"를 해결하였다.

10. 멀티모달 AI

🧒 멀티모달 = 여러 감각을 하나로

이미지·텍스트·소리를 다 이해하는 AI. 핵심 아이디어: 모두 같은 "벡터 공간"에 놓기.

여러 나라 말(이미지어, 글자어, 소리어)을 하나의 "번역기 언어"로 통일. 그 언어로는 고양이 사진 = "고양이"라는 글자 = "야옹" 소리.
  • CLIP: "이 사진이 '강아지 달리기'라는 설명과 얼마나 어울리나?" 계산기
  • GPT-4V / Claude 3: 사진 보여주면 설명하고 질문에 답하는 AI
  • Sora / Veo: 글 쓰면 영상을 만들어줌

그림 13. CLIP의 대조학습. 4억 쌍의 (이미지, 텍스트)를 공유 벡터공간에 매핑하여 제로샷 분류·검색·생성 조건 부여를 가능하게 했다.

10.1 CLIP과 대조 학습

Radford et al.(2021)의 CLIP은 4억 쌍의 (이미지, 텍스트)에 대해 InfoNCE 손실로 공동 임베딩 공간을 학습하였다: \[ \mathcal{L} = -\log \frac{\exp(\mathrm{sim}(I_i, T_i)/\tau)}{\sum_{j}\exp(\mathrm{sim}(I_i, T_j)/\tau)} \] 결과 모델은 제로샷 분류, 텍스트-이미지 검색, 그리고 Stable Diffusion의 텍스트 조건부여에서 핵심 구성요소가 되었다.

10.2 시각-언어 모델 (VLM)

GPT-4V, Claude 3, Gemini, LLaVA는 이미지를 LLM의 입력 공간으로 사영하는 어댑터를 두어 멀티모달 대화를 가능하게 한다. 전형적 구조: CLIP/SigLIP 비전 인코더 → MLP 또는 Q-Former → LLM.

10.3 텍스트-투-비디오와 세계 모델

Sora(2024), Veo, Kling 등은 잠재 확산과 Transformer 백본을 결합한 텍스트-투-비디오 모델이다. 이들이 암묵적 세계 모델을 학습하는지에 대한 논쟁이 있으며, 물리적 일관성은 여전히 활발한 연구 주제이다.

한 줄로 이해하기

강화학습(RL)은 "시행착오로 보상을 최대화하는 법을 배우는 AI"예요. 마치 강아지에게 간식을 주며 훈련시키는 것과 같죠. 멀티모달 AI는 텍스트·이미지·음성·영상 등 여러 종류의 데이터를 동시에 다루는 모델로, GPT-4o·Gemini가 대표적입니다.

왜 중요한가?

① 알파고가 이세돌 9단을 이긴 건 강화학습의 역사적 사건이었어요. ② 자율주행, 로봇 제어, 게임 AI 모두 강화학습을 사용합니다. ③ ChatGPT가 자연스럽게 답하도록 정렬한 RLHF도 강화학습이에요. ④ 멀티모달 AI 덕에 사진을 찍어 "이게 뭐야?"라고 물으면 음성으로 답변받는 일이 가능해졌습니다. ⑤ 시각장애인 보조기기 Be My Eyes·Seeing AI는 카메라 영상을 즉시 해설해줍니다.

🌱 핵심 용어

  • 에이전트: 행동을 결정하는 학습자
  • 환경: 에이전트가 상호작용하는 세계
  • 상태(state) / 행동(action) / 보상(reward): RL의 3대 요소
  • 정책(policy): "어떤 상황에서 어떤 행동을 할지"의 규칙
  • 탐험 vs 활용: 새로운 길을 시도할까, 알려진 좋은 길을 갈까
  • 멀티모달: 텍스트+이미지+음성 등 여러 모달리티를 함께 처리
  • CLIP: 이미지와 텍스트를 같은 공간에 매핑하는 모델

핵심 개념 풀어쓰기

강화학습을 비디오 게임에 비유해볼게요. 처음에는 마리오를 어디로 움직여야 할지 몰라 자꾸 죽어요(낮은 보상). 그런데 점프해서 코인을 먹으면 점수가 오르고(높은 보상), 구덩이에 빠지면 죽죠(음의 보상). 수만 번 플레이를 반복하면 "어떤 상황에서 어떻게 움직여야 점수를 최대화할 수 있는지" 자연스럽게 배웁니다. 이게 강화학습이에요. 멀티모달 AI는 "다재다능한 비서"라고 생각하면 됩니다. 사진을 보여주고 음성으로 질문하면 글로 답해주는, 인간처럼 여러 감각 정보를 동시에 처리하는 AI입니다.

💡 학습 TIP: OpenAI Gym의 CartPole 환경에서 직접 무작위 정책 → DQN → PPO 순으로 실험해보세요. 보상 그래프가 점점 우상향하는 모습을 보면 RL의 본질이 한순간에 와닿습니다. 멀티모달은 직접 ChatGPT 4o, Claude 3.5에 사진을 던져 다양한 질문을 해보는 게 가장 빠른 학습법이에요.

한 페이지 요약

  • RL = 시행착오 + 보상으로 정책 학습
  • 3대 알고리즘 패밀리: 가치 기반(DQN), 정책 기반(REINFORCE, PPO), Actor-Critic(A3C, SAC)
  • 대표 성과: AlphaGo, AlphaZero, MuZero, AlphaFold, ChatGPT(RLHF)
  • 멀티모달 = CLIP·BLIP·LLaVA·GPT-4o 같은 텍스트+이미지+음성 통합 모델
  • 응용: 로봇, 자율주행, 게임, 의료영상, 접근성 도구
중 · 응용

이론 깊이 들어가기

강화학습은 마르코프 의사결정 과정(MDP) ⟨S, A, P, R, γ⟩로 형식화됩니다. 에이전트는 정책 π(a|s)를 학습해 누적 할인 보상 G_t = Σ γ^k r_(t+k+1)의 기댓값을 최대화합니다. Bellman 방정식 V^π(s) = E[r + γV^π(s')]가 핵심이고, Q-learning은 Q*(s,a) = E[r + γ max_a' Q*(s',a')]를 부트스트래핑합니다. DQN(Mnih 2015)은 신경망으로 Q를 근사하여 Atari 49개 게임에서 인간 수준을 달성했습니다. Policy gradient(REINFORCE)는 ∇J = E[∇log π(a|s) · G_t]로 직접 정책을 업데이트하며, PPO(Schulman 2017)는 clipped objective로 안정성을 확보해 사실상 표준이 되었습니다. AlphaGo(Silver 2016)는 supervised learning + self-play RL + MCTS로 인간 챔피언을 이겼고, AlphaZero(2017)는 인간 데이터 없이도 같은 성능을 달성했습니다. 멀티모달은 CLIP(Radford 2021)이 이미지-텍스트 contrastive learning으로 4억 쌍을 학습해 zero-shot 분류를 가능케 했습니다. LLaVA, BLIP-2는 vision encoder를 LLM에 연결하는 어댑터(Q-Former, MLP projector) 구조이고, GPT-4o, Gemini는 처음부터 통합 토큰화로 학습됩니다.

핵심 개념과 코드

# Stable-Baselines3로 PPO 학습 (CartPole)
import gymnasium as gym
from stable_baselines3 import PPO

env = gym.make("CartPole-v1")
model = PPO("MlpPolicy", env, verbose=1, n_steps=2048,
            learning_rate=3e-4, gamma=0.99)
model.learn(total_timesteps=50_000)

obs, _ = env.reset()
for _ in range(500):
    action, _ = model.predict(obs, deterministic=True)
    obs, reward, terminated, truncated, _ = env.step(action)
    if terminated or truncated:
        obs, _ = env.reset()

대표 예제

💻 예제 1 — AlphaFold 2

단백질 구조 예측에 강화학습이 직접적이진 않지만, MSA + Evoformer + Structure Module의 end-to-end 학습은 RL의 정신을 이어받았습니다. CASP14에서 GDT_TS 92점으로 50년 묵은 생물학 난제를 해결, 2024 노벨화학상의 토대가 되었습니다.

💻 예제 2 — CLIP의 Zero-Shot

CLIP은 이미지와 텍스트를 같은 임베딩 공간에 매핑합니다. 분류기를 만들 때 별도 학습 없이 "a photo of a {label}" 텍스트와 이미지의 코사인 유사도로 zero-shot 분류가 가능합니다. ImageNet에서 ResNet-50과 비슷한 성능을 fine-tuning 없이 달성했습니다.

응용 사례

① 로봇 제어: Boston Dynamics, Tesla Optimus, Figure 01이 RL + sim-to-real로 실세계 동작 학습. ② 자율주행: Wayve, Comma.ai가 end-to-end RL/imitation learning 시도. ③ 추천 시스템: 컨텍스트 밴딧, off-policy RL로 클릭률·체류시간 최적화. ④ ChatGPT의 RLHF는 LLM에 RL을 적용한 가장 큰 상용 사례. ⑤ 멀티모달 의료: 방사선 영상 + 환자 기록을 함께 분석하는 GPT-4 Med, Med-PaLM-M.

⚠️ 자주 하는 실수: RL은 보상 함수 설계가 가장 어렵습니다. 보상 해킹(reward hacking)으로 의도치 않은 행동을 학습할 수 있어요. 예: 청소 로봇이 쓰레기를 숨기기. 또 sample efficiency가 낮아 시뮬레이터 없이는 비현실적이며, sim-to-real gap도 큽니다.
상 · 심화

전문가 레벨 이해

현대 RL의 이론적 토대는 dynamic programming(Bellman), 최적제어(Pontryagin), 그리고 stochastic approximation(Robbins-Monro)에 있습니다. Policy gradient theorem(Sutton 1999)은 ∇J(θ) = E_π[Q^π(s,a) ∇log π_θ(a|s)]로, 정책 매개변수에 대한 기대 누적 보상의 기울기를 직접 추정합니다. Actor-Critic은 advantage A(s,a) = Q(s,a) − V(s)로 분산 감소를 시도하고, GAE(Schulman 2016)는 λ-return으로 bias-variance trade-off를 제어합니다. PPO의 clipped surrogate objective L^CLIP = E[min(r_t A_t, clip(r_t, 1−ε, 1+ε) A_t)]는 단순하지만 강력하며, ChatGPT의 RLHF에 직접 사용됩니다. Offline RL(D4RL, Conservative Q-Learning)은 분포 외 행동의 over-estimation을 막는 것이 핵심 이슈입니다. World Model(Ha & Schmidhuber, Dreamer 시리즈)은 환경 모델을 학습하고 내부 simulation으로 정책을 학습하여 sample efficiency를 크게 높입니다. AlphaZero·MuZero(Schrittwieser 2020)는 plan-aware learning의 결정판입니다. 멀티모달은 contrastive(CLIP), generative(BLIP, GPT-4V), interleaved(Flamingo, IDEFICS), unified tokenization(GPT-4o, Chameleon, Gemini) 패러다임으로 진화했습니다. Vision Language Action Model(VLA: RT-2, OpenVLA)은 멀티모달 + 행동 토큰으로 로봇 정책을 LLM처럼 학습합니다.

수학적 기초

$$\text{Bellman Optimality: } Q^*(s, a) = \mathbb{E}[r + \gamma \max_{a'} Q^*(s', a')]$$ $$\text{Policy Gradient: } \nabla_\theta J(\theta) = \mathbb{E}_{\pi_\theta}[\nabla_\theta \log \pi_\theta(a|s) \cdot A^{\pi_\theta}(s, a)]$$ $$\text{PPO Clipped: } L^{CLIP}(\theta) = \mathbb{E}_t\left[\min(r_t(\theta) A_t, \text{clip}(r_t(\theta), 1-\epsilon, 1+\epsilon) A_t)\right]$$ $$\text{CLIP contrastive: } \mathcal{L} = -\frac{1}{2}\sum_i \log \frac{\exp(\langle v_i, t_i\rangle / \tau)}{\sum_j \exp(\langle v_i, t_j\rangle / \tau)} + (\text{symmetric})$$

최신 연구와 frontier

① RL for reasoning: DeepSeek-R1, OpenAI o1·o3는 GRPO/REINFORCE로 chain-of-thought를 직접 보상 학습. 수학·코딩에서 인간 전문가 수준. ② Embodied AI: RT-2, OpenVLA, RoboCat은 로봇에 LLM을 연결해 자연어 명령으로 작업 수행. ③ World Models: Genie 2, GameNGen, Sora가 인터랙티브 시뮬레이션 가능한 비디오 생성. ④ Agentic AI: Claude Computer Use, Operator, Manus는 멀티모달 LLM이 GUI를 직접 조작.

관련 분야 연결

  • 최적제어: HJB 방정식, LQR, MPC와 RL의 통합
  • 게임이론: Nash equilibrium, multi-agent RL, mean-field game
  • 신경과학: dopamine = TD error, basal ganglia = actor-critic
  • 인지과학: 의미 통합(grounding), 다감각 통합 이론
  • 로봇공학: SLAM, manipulation, sim-to-real transfer

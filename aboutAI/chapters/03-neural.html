5. 신경망과 딥러닝

🧒 신경망이란?

뉴런 = 스위치 하나. 입력 몇 개 받아서 그게 충분히 크면 "on", 작으면 "off". 이걸 수천만 개 쌓으면 고양이 사진도 알아본다.

뉴런 = 소문 퍼뜨리는 사람. 친구 3명 말을 종합해서 판단 후 다음 사람에게 전달. 이 체인이 수백 층.
역전파 = 게임에서 졌을 때 "어느 단계부터 잘못됐지?" 거꾸로 추적하며 고치기. AI가 학습하는 방법.
ReLU 활성함수 = "음수는 그냥 0으로 해버리자" (양수만 살림). 단순한데 엄청 잘 먹힌다.

💡 CNN은 이미지용 (작은 윈도우로 스캔하며 패턴 찾기 = 돋보기). LSTM은 시퀀스용 (기억 게이트로 중요한 것만 간직).

5.1 다층 퍼셉트론 (MLP)

  • Sigmoid: $\sigma(x) = 1/(1+e^{-x})$. 포화로 인한 기울기 소실(vanishing gradient) 문제.
  • Tanh: $\tanh(x)$. 0-중심이나 여전히 포화함.
  • ReLU(Nair & Hinton, 2010): $\max(0, x)$. 희소 활성화와 비포화로 표준이 됨.
  • GELU(Hendrycks & Gimpel, 2016): $x\Phi(x)$, 여기서 $\Phi$는 표준정규 CDF. Transformer에서 주로 사용.
  • SwiGLU(Shazeer, 2020): $\text{Swish}(xW) \odot (xV)$. 현재 최신 LLM의 기본 활성함수.
정리 5.1 (보편근사정리, Cybenko 1989; Hornik 1991). 단일 은닉층 MLP는 유한한 뉴런 수로 $\mathbb{R}^n$의 임의의 컴팩트 부분집합 위에서 연속함수를 임의의 정밀도로 근사할 수 있다.

그러나 이 정리는 존재성만을 보장할 뿐, 학습 가능성이나 효율적 표현 크기를 보장하지 않는다. 깊이(depth)는 폭(width)보다 지수적으로 효율적일 수 있음이 알려져 있다(Telgarsky, 2016).

5.2 역전파 (Backpropagation)

손실 $\mathcal{L}$의 파라미터 $\mathbf{W}^{(l)}$에 대한 그래디언트는 연쇄법칙으로 계산된다: \[ \frac{\partial \mathcal{L}}{\partial \mathbf{W}^{(l)}} = \frac{\partial \mathcal{L}}{\partial \mathbf{h}^{(l)}} \cdot \frac{\partial \mathbf{h}^{(l)}}{\partial \mathbf{W}^{(l)}} \] 여기서 $\delta^{(l)} = \partial \mathcal{L}/\partial \mathbf{h}^{(l)}$을 오차 신호라 부르며, 다음 점화식을 따른다: \[ \delta^{(l)} = (\mathbf{W}^{(l+1)})^\top \delta^{(l+1)} \odot \sigma'(\mathbf{z}^{(l)}) \] 역전파는 자동 미분(automatic differentiation)의 역방향 모드(reverse-mode)에 해당하며, 현대 프레임워크(PyTorch, JAX)는 이를 계산 그래프(computational graph) 위에서 자동화한다.

5.3 합성곱 신경망 (CNN)

CNN(LeCun et al., 1989; 1998)은 이미지의 공간 구조에 맞춘 귀납 편향을 내장한다. 핵심 연산은 이산 합성곱: \[ (\mathbf{I} * \mathbf{K})(i, j) = \sum_{m}\sum_{n}\mathbf{I}(i+m, j+n)\mathbf{K}(m, n) \] CNN의 세 가지 특성:

  1. 지역 연결성: 각 뉴런은 입력의 지역적 패치만 본다(수용장, receptive field).
  2. 파라미터 공유: 동일한 커널이 공간 전체에서 재사용됨 → 이동 등변성.
  3. 풀링: 공간 해상도 축소와 이동 불변성 강화.

주요 아키텍처: LeNet-5(1998), AlexNet(2012), VGG(2014), GoogLeNet/Inception(2014), ResNet(2015)은 $\mathbf{h}_{l+1} = \mathbf{h}_l + F(\mathbf{h}_l)$의 항등 지름길을 통해 매우 깊은 네트워크의 훈련을 가능하게 했다. ConvNeXt(2022)는 Transformer의 설계 원칙을 CNN에 적용하여 경쟁력을 회복시켰다.

5.4 순환 신경망 (RNN, LSTM, GRU)

RNN은 시퀀스 $\mathbf{x}_1, \dots, \mathbf{x}_T$를 은닉상태 $\mathbf{h}_t$로 처리한다: \[ \mathbf{h}_t = \sigma(\mathbf{W}_{hh}\mathbf{h}_{t-1} + \mathbf{W}_{xh}\mathbf{x}_t + \mathbf{b}) \] 그러나 단순 RNN은 시간을 통한 역전파(BPTT)에서 기울기 소실/폭발 문제를 겪는다. LSTM(Hochreiter & Schmidhuber, 1997)은 게이팅 메커니즘으로 이를 완화한다: \[ \begin{aligned} \mathbf{f}_t &= \sigma(\mathbf{W}_f[\mathbf{h}_{t-1}, \mathbf{x}_t]) \quad (\text{망각 게이트})\\ \mathbf{i}_t &= \sigma(\mathbf{W}_i[\mathbf{h}_{t-1}, \mathbf{x}_t]) \quad (\text{입력 게이트})\\ \mathbf{o}_t &= \sigma(\mathbf{W}_o[\mathbf{h}_{t-1}, \mathbf{x}_t]) \quad (\text{출력 게이트})\\ \mathbf{c}_t &= \mathbf{f}_t \odot \mathbf{c}_{t-1} + \mathbf{i}_t \odot \tanh(\mathbf{W}_c[\mathbf{h}_{t-1}, \mathbf{x}_t])\\ \mathbf{h}_t &= \mathbf{o}_t \odot \tanh(\mathbf{c}_t) \end{aligned} \] 그러나 RNN 계열은 본질적으로 순차적이므로 병렬화가 제한되며, 긴 의존성에 약하다. 이 한계는 Transformer의 등장 동기가 되었다.

5.5 표현학습 (Representation Learning)

딥러닝의 가치는 수작업 특징 공학을 자동화된 특징 학습으로 대체한 것에 있다. Bengio et al.(2013)의 "Representation Learning" 리뷰는 좋은 표현의 조건으로 매니폴드 가설, 분산 표현, 위계적 추상화를 제시하였다. 자기지도학습(SimCLR, MoCo, DINO, MAE)은 레이블 없이 강력한 표현을 학습하여, 파운데이션 모델 패러다임의 토대를 제공했다.

한 줄로 이해하기

신경망은 사람 뇌의 뉴런을 흉내 낸 수학 모델이에요. 입력값에 가중치를 곱하고 더한 뒤 비선형 함수를 통과시키는 작은 단위(퍼셉트론)를 여러 층으로 쌓아 복잡한 패턴을 학습합니다. "딥러닝"의 "딥(deep)"은 바로 이 층이 여러 개라는 뜻이에요.

왜 중요한가?

① 스마트폰의 얼굴 인식 잠금 해제는 CNN(합성곱 신경망)이 사진의 픽셀 패턴을 학습해 만든 결과입니다. ② 음성 비서(시리, 빅스비)가 말을 알아듣는 건 RNN/Transformer 신경망이 음성 신호를 텍스트로 변환하기 때문이에요. ③ 카드사가 이상거래를 자동 탐지하는 사기 방지 시스템도 신경망의 분류 능력 덕분입니다.

🌱 핵심 용어

  • 뉴런(노드): 입력에 가중치를 곱해 더하고 활성화 함수를 적용하는 작은 계산 단위
  • 층(layer): 같은 역할의 뉴런들이 한 줄로 모인 구조
  • 활성화 함수: ReLU, Sigmoid, Tanh — 비선형성을 더해주는 함수
  • 역전파: 오차를 거꾸로 전파해서 가중치를 조금씩 고치는 알고리즘
  • CNN: 이미지에 특화된 신경망 (합성곱+풀링)
  • RNN/LSTM: 시계열·문장 같은 순서 데이터에 특화된 신경망

핵심 개념 풀어쓰기

신경망을 거대한 회사 조직에 비유해볼게요. 입력층은 신입사원이 외부 정보를 받는 곳이고, 은닉층들은 중간 관리자들이 정보를 가공·요약하는 곳, 출력층은 최종 결정을 내리는 임원이에요. 학습 과정은 "임원이 잘못된 결정을 내렸을 때 누구의 책임인지"를 거꾸로 추적해서 각자 일하는 방식을 미세 조정하는 거예요. 이게 역전파(backpropagation)입니다. CNN은 이미지의 작은 영역을 훑는 "현미경"이라고 생각하면 돼요. 처음에는 모서리·색깔 같은 단순 특징을, 층이 깊어질수록 눈·코·입 같은 복잡한 특징을 알아봅니다.

💡 학습 TIP: 신경망을 처음 배울 땐 직접 종이에 2-3-1 구조의 작은 네트워크를 그리고, 손으로 forward pass와 backward pass를 한 번 계산해보세요. 그러고 나면 PyTorch나 TensorFlow의 추상화가 한순간에 이해됩니다.

한 페이지 요약

  • 신경망 = 뉴런(가중합 + 활성화)을 층층이 쌓은 함수 근사기
  • 학습 = 손실(오차)이 줄어드는 방향으로 가중치 업데이트
  • 핵심 알고리즘은 1986년의 역전파(backpropagation)
  • CNN(이미지), RNN/LSTM(순서 데이터), Transformer(어텐션 기반)가 3대 가족
  • 층이 깊을수록 추상도가 높은 특징을 자동으로 발견
중 · 응용

이론 깊이 들어가기

다층 퍼셉트론(MLP)은 universal approximation theorem(Cybenko 1989; Hornik 1991)에 의해 임의의 연속 함수를 임의 정밀도로 근사할 수 있습니다. 다만 폭과 깊이가 충분해야 하죠. 활성화 함수는 비선형성을 부여해 단순한 선형결합 이상의 표현력을 만듭니다. ReLU(x) = max(0, x)는 sigmoid의 vanishing gradient 문제를 해결해 깊은 네트워크 훈련을 가능케 했습니다. 학습 알고리즘은 SGD가 표준이지만 실전에서는 Adam, AdamW가 압도적으로 쓰입니다. 이는 1차 모멘트와 2차 모멘트의 추정으로 적응적 학습률을 제공해 수렴 속도를 높입니다. 합성곱 신경망(CNN)의 핵심은 매개변수 공유(parameter sharing)와 평행이동 등변성(translation equivariance)입니다. 동일 필터를 이미지 전체에 적용하므로 위치에 무관한 패턴을 학습합니다. 풀링은 공간 해상도를 줄이며 작은 변형에 강건성을 줍니다. RNN은 hidden state h_t = f(h_(t−1), x_t)로 시간 의존성을 모델링하지만 vanishing gradient에 취약합니다. LSTM(1997)과 GRU(2014)는 gating으로 이를 해결합니다.

핵심 개념과 코드

import torch
import torch.nn as nn

class MLP(nn.Module):
    def __init__(self):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(784, 256), nn.ReLU(),
            nn.Linear(256, 128), nn.ReLU(),
            nn.Linear(128, 10)
        )
    def forward(self, x):
        return self.net(x)

model = MLP()
opt = torch.optim.AdamW(model.parameters(), lr=1e-3)
loss_fn = nn.CrossEntropyLoss()
# x: (B, 784), y: (B,)
logits = model(x)
loss = loss_fn(logits, y)
loss.backward()
opt.step(); opt.zero_grad()

대표 예제

💻 예제 1 — ResNet의 잔차 연결

ResNet은 h(x) = F(x) + x 형태로 입력을 출력에 더해줍니다. 이는 그래디언트가 항등 경로(identity path)를 통해 직접 흘러서 vanishing gradient를 완화하고, 1000층 이상의 네트워크 훈련을 가능하게 했습니다. ImageNet 2015 우승, 이후 모든 비전 모델의 표준이 되었습니다.

💻 예제 2 — Batch Normalization

BN(2015)은 미니배치의 평균·분산으로 활성화를 정규화한 뒤 학습 가능한 γ, β로 재스케일합니다. 내부 공변량 변화(internal covariate shift)를 줄여 학습률을 크게 잡을 수 있고, 정칙화 효과도 있습니다. Layer Norm, Group Norm으로 변종이 발전했습니다.

응용 사례

① 자율주행의 인식 모듈은 EfficientNet, YOLO, Detectron2 같은 CNN 계열로 차량/보행자/표지판을 검출합니다. ② 의료 영상의 폐결절·당뇨망막증 자동 진단은 ResNet, U-Net으로 픽셀 단위 분할까지 수행합니다. ③ 추천 시스템은 user-item 임베딩 + MLP로 클릭률을 예측하며, YouTube의 candidate generation도 이 구조입니다. ④ 음성 인식은 Conformer(CNN+Transformer 하이브리드)가 SOTA를 차지합니다.

⚠️ 자주 하는 실수: 학습률을 너무 크게 잡으면 발산하고, 너무 작으면 수렴이 느립니다. lr finder나 cosine schedule을 활용하세요. 또 Batch Norm을 사용할 땐 batch size가 너무 작으면 통계가 부정확해지므로 GroupNorm/LayerNorm을 고려해야 합니다.
상 · 심화

전문가 레벨 이해

딥러닝의 일반화 미스터리는 고전적 PAC 이론으로 설명되지 않습니다. 매개변수 수가 학습 데이터 수를 훨씬 초과해도 잘 일반화되는 현상은 implicit regularization, neural tangent kernel(NTK, Jacot 2018), feature learning theory(Yang & Hu 2021) 등으로 부분 설명됩니다. NTK 영역에서는 무한 폭 신경망이 가우시안 프로세스로 수렴하며, 학습 동역학이 선형화됩니다. 그러나 실제 유한 폭 네트워크는 NTK를 넘어 feature learning이 일어나며, 이것이 표현 학습의 본질입니다. 최적화 측면에서 SGD의 노이즈가 평탄한 최소점을 선호한다는 것은 Keskar et al.(2017)와 후속 연구가 보였고, 최근에는 sharpness-aware minimization(SAM, Foret 2021)이 명시적으로 sharpness를 페널티로 부과합니다. CNN의 인덕티브 바이어스는 locality, translation equivariance, hierarchy인데, ViT(Vision Transformer, 2020)는 이런 바이어스 없이도 충분한 데이터(JFT-300M)에서는 CNN을 능가했습니다. 이는 "데이터가 많으면 인덕티브 바이어스가 약한 모델이 유리하다"는 bitter lesson(Sutton)의 직접적 사례입니다. RNN의 한계는 long-range 의존성 학습 시 그래디언트 폭발/소실에서 옵니다. LSTM의 gating은 부분 해법이며, 더 근본적 해결책이 self-attention입니다.

수학적 기초

$$\text{ReLU 그래디언트: } \frac{\partial \text{ReLU}(z)}{\partial z} = \mathbb{1}[z > 0]$$ $$\text{역전파 (chain rule): } \frac{\partial L}{\partial W^{(l)}} = \delta^{(l)} (a^{(l-1)})^T,\quad \delta^{(l)} = (W^{(l+1)})^T \delta^{(l+1)} \odot \sigma'(z^{(l)})$$ $$\text{Convolution: } (f * g)(t) = \sum_{\tau} f(\tau) g(t - \tau)$$ $$\text{LSTM gates: } f_t = \sigma(W_f [h_{t-1}, x_t] + b_f),\; i_t, o_t \text{ 동형}$$

최신 연구와 frontier

① State Space Models: Mamba(Gu & Dao, 2023), S4 등은 선형 시간 복잡도로 long-context 모델링을 시도하며 Transformer의 대안으로 부상 중입니다. ② Mixture of Depths/Experts: 토큰별로 다른 깊이/전문가를 활용하는 동적 계산 그래프 연구. ③ μP (Maximal Update Parametrization): Yang et al.은 모델 폭을 키워도 하이퍼파라미터가 전이되는 정규화를 제안, 대규모 학습의 안정성을 높였습니다.

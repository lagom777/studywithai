2장. CSS와 레이아웃

2.1 CSS 기초

CSS(Cascading Style Sheets)는 HTML 요소의 모양, 레이아웃, 애니메이션을 정의합니다. "캐스케이딩"이란 여러 스타일 규칙이 우선순위에 따라 적용되는 것을 의미합니다.

선택자(Selectors)

/* 요소 선택자 */ p { color: blue; } /* 클래스 선택자 */ .highlight { background: yellow; } /* ID 선택자 */ #header { font-size: 24px; } /* 조합 선택자 */ div > p { margin: 10px; } div p { padding: 5px; } /* 속성 선택자 */ input[type="email"] { border: 2px solid blue; } /* 의사 클래스 */ a:hover { color: red; } li:first-child { font-weight: bold; }

특이성(Specificity)

특이성은 어떤 CSS 규칙이 적용될지 결정합니다. 점수가 높을수록 높은 우선순위를 가집니다:

  • ID 선택자: 100점
  • 클래스, 속성, 의사 클래스: 10점
  • 요소 선택자: 1점

2.5 애니메이션과 트랜지션

CSS 애니메이션과 트랜지션으로 부드러운 시각 효과를 만들 수 있습니다. @keyframes를 사용하여 복잡한 애니메이션을 정의합니다.

@keyframes slideIn { from { opacity: 0; transform: translateX(-50px); } to { opacity: 1; transform: translateX(0); } } .element { animation: slideIn 1s ease-out forwards; }

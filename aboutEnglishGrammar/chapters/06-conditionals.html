가정법:
현실과 비현실의 경계

Chapter 6 — Conditionals

Zero / First / Second / Third / Mixed Conditional

가정법의 4가지 유형

영어 가정법은 조건의 실현 가능성에 따라 4가지로 분류됩니다. 한국어는 가정법과 직설법의 구분이 문법 형태로 명확하지 않지만, 영어는 동사 시제로 확실히 구분합니다.

🌐 현실 ↔ 비현실의 차원 — 드래그로 탐색

Zero Conditional — 항상 참인 사실

If + 현재, 현재: If you heat water to 100°C, it boils.

First Conditional — 실현 가능한 미래

If + 현재, will + V: If it rains, I will stay home.

Second Conditional — 현재/미래 비현실

If + 과거, would + V: If I were rich, I would travel the world. (실제론 부자 아님)

Third Conditional — 과거 비현실 (후회/가정)

If + 과거완료, would have + p.p.: If I had studied, I would have passed.

혼합 가정법 (Mixed Conditional)

If + 과거완료(과거 조건), would + V(현재 결과)
"If I had taken that job, I would be in Paris now."

가정법의 도치

if 생략 → 도치

If I were you... → Were I you...
If I had known... → Had I known...
If it should happen... → Should it happen...

📚 난이도별 학습

하 · 입문

한 줄로 이해하기

가정법은 "만약 ~라면"입니다. 실현 가능하면 1st, 불가능하면 2nd, 과거에 대한 후회면 3rd!

🌱 핵심 구분

  • "내일 비 오면 집에 있을게" → 1st (가능)
  • "내가 새라면 날 텐데" → 2nd (불가능)
  • "공부했더라면 합격했을 텐데" → 3rd (과거 후회)
💡 학습 TIP: 2nd conditional에서 be동사는 주어 관계없이 were를 씁니다: "If I were you..." (am이 아님)
중 · 응용

wish 가정법

현재 아쉬움: I wish I were taller. (2nd)
과거 후회: I wish I had studied. (3rd)
미래 바람: I wish it would stop raining.

as if / as though

He talks as if he knew everything. (사실 모름, 2nd) vs. as if he had known (과거, 3rd)

⚠️ 주의: unless = if...not: Unless you hurry, you'll be late. = If you don't hurry...
상 · 심화

가정법의 양상(Modality) 이론

가정법(Subjunctive Mood)은 양상 논리(Modal Logic)와 밀접합니다. 가능세계 의미론(Possible Worlds Semantics)에서 조건문은 접근가능 세계(Accessible Worlds)에서의 진리치로 분석됩니다. Stalnaker-Lewis 분석: 조건문 "If A, then B"는 A가 성립하는 가장 가까운 가능세계에서 B가 성립함을 의미합니다. 반사실적 조건문(Counterfactual Conditionals)은 이 분석의 핵심 영역입니다.

형식 표현

□(A → B): A인 모든 접근 세계에서 B 성립
◇A: A가 성립하는 접근 세계가 존재
🎓 도전 과제: "If Caesar hadn't crossed the Rubicon, Rome might have remained a republic."를 가능세계 의미론으로 분석하고, 이것이 왜 반사실적 조건문인지 설명하세요.

How to use this site

이 사이트는 '심리학이 뭐지?' 하는 분부터 '교양서 한두 권 읽어봤지만 체계적으로 잡고 싶다'는 분까지 쓸 수 있게 만든 입문용 학습 페이지입니다.

학습 방법

왼쪽 목차에서 관심 있는 챕터를 선택하세요. 순서대로 봐도 좋고, 관심 있는 주제만 골라 봐도 됩니다.
각 챕터는 기본 정의 → 그림으로 보는 원리 → 일상 예시 → 흔한 오해 순서로 구성되어 있습니다.
보라색 박스는 용어 정의, 노란색 박스는 실생활 예시, 초록색 박스는 공부 팁, 주황색 박스는 주의할 점입니다.
마지막에 확인 퀴즈가 있으니, 한 챕터 마칠 때마다 풀어보며 복습해 보세요.
공부 팁 심리학은 용어가 많아 보이지만, "왜 이 개념이 나왔는지"를 함께 이해하면 훨씬 쉽게 외워집니다. 각 개념마다 "이걸 몰랐을 때 사람들이 어떤 오해를 했을까?"를 떠올려 보세요.
